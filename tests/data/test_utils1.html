機能
Features

地図から一目で

地図から建物や教室を検索することができます。もう広いキャンパスで迷うことはありません。新歓やセミナーなど様々なイベントが地図上に表示されています。スマートフォンを片手に興味の赴くイベントに足を運んでみましょう。

充実した情報

京大マップは学内の建物や教室の情報を網羅しており、学内で行われるあらゆるイベント情報がアプリ内で共有されています。吉田・桂・宇治の3キャンパスに対応し、アプリ内で切り替えることが可能です。

新たな告知ツール

京大マップでは誰でも手軽にイベントを投稿することができます。投稿されたイベントをTwitterやFacebookで共有することもできます。大量のビラを配布することなく、多くの学生にイベントの告知ができます。

ラインナップ
Lineup

京大マップ for iOS

iOS に最適化された機能と操作性。最新の iPhone と Apple Watch にも対応。 iOS 8.0 以上の端末でご利用いただけます。

京大マップ for Android

Android 4.0 以上に対応。Material Design を採用してより使いやすくなりました。 多くの Android 端末でご利用いただけます。

京大マップ Web

スマートフォンや PC, Mac 等の Web ブラウザからご利用いただけます。 スマートフォンをお持ちでない方や、アプリを持っていない方に情報を伝えるのに便利です。
